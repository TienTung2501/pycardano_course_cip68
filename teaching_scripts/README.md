# Teaching Scripts (CIP-68 Dynamic NFT Platform)

Bộ kịch bản giảng dạy chi tiết theo 4 phần của khóa học.

## Mục tiêu cuối buổi
- Hiểu CIP-68 (Dynamic NFT) và CIP-67 labels.
- Đọc/giải thích được smart contract Aiken (minting policy + spending validator).
- Chạy được off-chain code (mint/update/burn) trên Preprod.
- Demo end-to-end dApp: frontend (Next.js) + backend (FastAPI/PyCardano) + CIP-30 browser wallet.

- Phần 1 (Slide lý thuyết): `PHAN-1-ly-thuyet-slide.md`
